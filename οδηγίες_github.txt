##Οδηγίες για τη χρήση του Github##
===================================

1/ Στήνεις το git από εδώ:  https://git-scm.com/

2/ Ανοίγεις το command prompt στα windows

3/ Πληκτρολογείς:  cd Documents  

4/ Πληκτρολογείς:  git clone  https://github.com/dimitristsoump/Oil_Spill_Optimization.git

5/ Όταν τελειώσει, στα Documents θα σου έχει δημιουργηθεί ένα folder Oil_Spill_Optimization που θα περιέχει όλα τα αρχεία του Github


6/ Αν κάνεις αλλαγή σε αρχείο πρέπει να κάνεις:

cd Documents

cd Oil_Spill_Optimization

git add .

git commit -m  "Εδώ γράφεις τι άλλαξες με κατανοητό και σύντομο τρόπο"

git push origin master              

+ συμπληρώνεις Username, κωδικό αν το ζητήσει

πάντα στο command prompt εννοείται 

7/ Αν έχει ανεβάσει άλλος κάτι πρώτα και εσύ δεν το έχεις και θες να το κατεβάσεις ή θες να κάνεις κάποια αλλαγή, γράφεις:

cd Documents

cd Oil_Spill_Optimization

git pull origin master

+βλ. 6 αν θες να ανεβάσεις κάτι μετά (από τη 2η εντολή και κάτω)

8/ Στη σελίδα του github αν πατήσεις εκεί που λέει  (# commits όπου #=1,2,3,...)   σου βγάζει τις παλιές version
τις οποίες μπορείς να ξαναδείς και να κατεβάσεις αν δεν συμφωνείς με μία αλλαγή


9/ Τέλος, βλέπε το README γιατί εκεί θα ενημερώνουμε ο ένας τον άλλο για το project (αλλαγές που έγιναν, πράγματα που ανέβηκαν, πράγματα που τελείωσαν). Μπορείς να το κάνεις edit και εσύ από το github. 

10/Καλή εφαρμογή για να τσεκάρεις ενημερώσεις στο repo από το κινητό είναι το fasthub
